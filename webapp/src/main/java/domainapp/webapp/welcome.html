Apache Isis™ es una plataforma que permite el rapido desarrollo de aplicaciones "domain-driven" en Java.

Esta Aplicacion fue generada usando Apache Isi
Proyecto de gestion del Mantenimiento de una Central Hidroelectrica tipo,
que permite la alta, baja, modificacion y eliminacion de los datos necesario para la misma,
como ser Tecnicos, Unidades de Mantenimiento, Insumos, y para el seguimiento de las Fichas de Control de Mantenimiento.
La aplicaicon tambien cuenta con un apartado de Estadisticas (Extensible) y de Reportes (Extensible).
Proyecto desarrollado por los Analistas: Manuel Romero, Franco Chidichimo y Juan Manuel Machado.
Tutor del Proyecto: Emiliano Pereyra
